# AWS EC2 CLI Management Tool

This project provides a **Python-based CLI tool** to manage AWS EC2 instances using the **AWS CLI** under the hood.

## Features

* Manage EC2 instances with the following operations:

  * `start`
  * `stop`
  * `reboot`
  * `terminate`
* Authenticates to AWS by exporting credentials as environment variables.
* Executes AWS CLI commands directly from Python.
* Monitors instance state transitions to ensure operations complete successfully.
* Prints detailed information for each instance:

  * **Instance ID**
  * **Name tag**
  * **State**
  * **Private IP**
  * **Public IP**

## Benefits

* Learn **AWS automation** without needing `boto3`.
* Understand **CLI integration** in Python scripts.
* Manage the **full lifecycle of EC2 instances** with simple commands.
